# ITMD 361 – Project 2: Adding CSS

This repository contains Project 2 for ITMD 361: Fundamentals of Web Development.

The goal of this project is to expand the HTML-only website created in Project 1 by adding external CSS to control layout, typography, and visual presentation while maintaining valid HTML structure.

## Project Overview

The website contains three pages:

- Home (index.html)
- Resume (resume.html)
- About (about.html)

All pages share a consistent navigation structure and layout while also having page-specific styling.

## Technologies Used

- HTML5 (semantic structure)
- CSS3 (external stylesheets)
- Google Fonts
- W3C HTML and CSS validation

## Project Requirements Implemented

- External CSS styling applied to all pages
- One CSS reset file and one primary stylesheet
- Centered horizontal navigation consistent across all pages
- At least five shared style elements across the website
- At least five unique style elements on each individual page
- One imported font and one browser-native fallback font
- Responsive layout using relative units (rem, %, vw)
- One fluid layout page and one fixed-width layout page
- Valid HTML and CSS confirmed through W3C validation
